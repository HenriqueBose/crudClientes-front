Projeto CRUD Clientes

-Realizado ultilizando:
Front-end: Angular 12, com o uso da biblioteca de UI PrimeNG
Back-end: SpringBoot 2.4.4
Persistencia de dados: Hibernate 5.4.32 com JPA
Banco: Postgres



Neste projeto é possível cadastrar, editar, deletar e visualizar clientes com as seguintes informações:
Nome, CPF, CEP, Cidade, Logradouro, Bairro, UF, Complemento, multiplos Emails e multiplos telefones
Existe uma autenticação de roles de usuário usando OAUTH2 atravéz do JWT, impedindo que o usuário comum acesse métodos que não sejam de leitura, essa validação se aplica tanto no nível de front-end quando no de back-end, ultilizando métodos de Authorization Server e CORS
